Repo: dynamic_programming
=========================

C++ implementation of some common dynamic programming problems.

Theory part video lectures are available at 
https://www.youtube.com/watch?v=lVR2u9lsxl8&list=PLdo5W4Nhv31aBrJE1WS4MR9LRfbmZrAQu

This Repo contains solutions for the following problems
=======================================================

1. Fibonacci series
2. Longest common substring
3. Longest common subsequence
4. Subsetsum (Whether exists or not)
5. Traveling salesman problem.
6. 0/1 Knapsack problem
7. Coin change - Total number of solutions for the given sum
8. Coin change - Minimum number of coins required for the given sum
